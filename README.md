# SPM Question Intelligence

SPM 历年题目趋势、题型覆盖和复习优先级分析原型。

## Phase 1 已完成

- 多科目筛选
- 年份、Paper、来源筛选
- 历年题目记录表
- 章节年度热度图
- 高频考点与复习优先级
- 透明评分模型
- 教师验证状态
- 手机与桌面响应式界面

## 运行方式

这是纯静态网页，直接打开 `index.html` 即可运行。

## GitHub Pages

在仓库进入：

`Settings → Pages → Build and deployment → Deploy from a branch`

选择：

- Branch: `main`
- Folder: `/ (root)`

保存后网页地址通常为：

`https://darkgeneration77-spec.github.io/spm-question-intelligence/`

## 评分说明

预测分数代表复习优先级，不代表真实出题概率。当前原型综合：

- 近期正式真题频率
- 课程重要度
- 覆盖缺口
- Trial 共识
- 题型适配
- 技能重复
- 教师审核
